# "Micro-Internship Hub" ЖИ-Платформасын Жасау Документациясы

## 1. Жобаның Мақсаты мен Мәселесі
**Мәселе:** Жұмыс берушілер «тәжірибесі бар» мамандарды іздейді, ал жаңа бітірген жастарда ол жоқ. Бұл "тұйық шеңбер" мәселесі көптеген жастардың жұмысқа орналасуына кедергі келтіреді.
**Шешім:** Жастарға арналған «Micro-Internship Hub» платформасы. Студенттер ЖИ (Жасанды Интеллект) көмегімен нақты компаниялардың кішігірім кейстерін шешіп, өздеріне «цифрлық портфолио» жинайды.

## 2. Қолданылған Технологиялар мен ЖИ Құралдары
Бұл жоба толықтай **No-Code (код жазбай)** принциптеріне негізделіп, ЖИ-агенттерді (Agentic AI) басқару арқылы жасалды.

* **Frontend:** React JS (Vite), CSS (Glassmorphism стилі)
* **Backend & DB:** Supabase (Auth, PostgreSQL)
* **Хостинг:** Vercel (Автоматты CI/CD)
* **ЖИ-Бағдарламашы:** Google Gemini Advanced (барлық кодты генерациялау және қателерді жөндеу)
* **Платформадағы ЖИ-Тәлімгер:** OpenAI API (GPT-4o моделі)
* **Дизайн мен Визуал:** AI Image Generator (DALL-E 3 / Imagen) логотип пен макет жасау үшін.

## 3. Платформаны Құрудың Қадамдық Процесі

### 1-Қадам: Визуалды Дизайн және UI/UX (AI Generation)
Алдымен платформаның стилі анықталды. Заманауи трендтерге сай мөлдір (Glassmorphism) дизайн таңдалды. ЖИ сурет генераторлары арқылы платформаның басты бетінің визуалды макеті (`platform_ui_mockup.png`) және жоба логотипі (`micro_internship_logo.png`) жасалды.

### 2-Қадам: Frontend Прототипін Құру (React)
Gemini ЖИ-агентінің көмегімен `npx create-vite` командасы арқылы React жобасы құрылды. Басты беттің (Landing Page) HTML/CSS құрылымы жазылып, жобаның толыққанды интерфейсі дайындалды. Формалар мен батырмалар интерактивті түрге келтірілді.

### 3-Қадам: Backend және Деректер Қорын Қосу (Supabase)
Жобаны динамикалық ету үшін Supabase бұлтты жүйесі қолданылды:
1. **Authentication:** Пайдаланушыларды тіркеу және логин жүйесі қосылды.
2. **Database:** Бизнес өкілдері тапсырмалар жариялауы үшін `tasks` кестесі (table) құрылды.
3. Қауіпсіздік үшін RLS (Row Level Security) саясаттары орнатылды.

### 4-Қадам: OpenAI (GPT-4o) Интеграциясы
Платформаның ең басты ерекшелігі — кіріктірілген ЖИ-тәлімгер. 
* Студент кез келген тапсырманы таңдаған кезде, оң жақта чат терезесі ашылады. 
* Чат OpenAI серверімен (gpt-4o моделі) тікелей байланысқан.
* ЖИ студентке тапсырманы орындап бермейді, тек **бағыт-бағдар беретін ментор** (тәлімгер) ретінде жұмыс істеуге бапталған (System Prompt арқылы шектелген).

### 5-Қадам: Автоматты Түрде Жариялау (GitHub & Vercel)
Жобаның барлық файлы жергілікті (local) Git репозиторийіне жинақталып, кодсыз әдіспен GitHub-қа (Disaaaaaaa/micri) жүктелді. Одан кейін Vercel платформасына қосылып, интернетте барлық адамға қолжетімді сілтемеге (`micri.vercel.app`) шығарылды.

## 4. Платформаның Негізгі Функциялары
1. **Кіру/Тіркелу:** Бизнес өкілдері мен студенттерге арналған қауіпсіз тіркелу жүйесі.
2. **Тапсырма Жариялау:** Кәсіпкерлер тақырыпты, бағаны, сипаттаманы жазып жаңа тапсырмалар қоса алады.
3. **Тапсырмалар Тақтасы:** 9 түрлі бағыттағы (IT, Дизайн, Аналитика, Маркетинг) шынайы тапсырмалар тізімі (MOCK_TASKS & Supabase DB).
4. **ЖИ-Чат (AI Assistant):** Тапсырманы орындау барысында нақты уақыт режимінде көмек беретін ақылды көмекші.

## 5. Сыни Рефлексия және ЖИ Этикасы
Бұл жобаны жасау барысында **Этикалық мәселе** басты назарда болды. 
"Студенттер ЖИ-ды қолданып, жұмысты өздері істемей қоймай ма?" деген сұрақ туындайды. Осыны болдырмау үшін платформадағы ЖИ тек кеңесші ретінде бағдарламаланған. Ол дайын жауап бермейді, студентті ізденіске итермелейтін сұрақтар қояды. 
Болашақта бұл платформаға "ЖИ-Эвалюатор" (студент жұмысын автоматты бағалайтын жүйе) және ойын элементтерін (Геймификация) қосу жоспарлануда.
